# Claim Boundary

This package proposes and specifies an architecture. It does not report a successful stable-plastic training run.

Supported claims:

- the current v2.3 retention objective and deployed posterior are structurally mismatched;
- a backward-compatible context-memory interface can be specified without dataset-specific branches;
- the proposed ablation and regression plan is executable and falsifiable.

Unsupported claims:

- the redesign improves MMALS accuracy or retention;
- stable-plastic memory solves catastrophic forgetting;
- the adaptive gate generalizes across datasets;
- positive backward transfer.
